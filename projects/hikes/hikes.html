Hikes

For the last couple of years, I have been recording our hikes around the country. My goal is to combine SQL API data access, data manipulation using Pandas, and Plotly graphing tools.
Check all my graphs and data in here.
All my graphs are interactive! You can hide each hike by clicking on it or double-click on one to select it.

My next step is to connect Fitbit's API with these hikes and track my data such as steps, heart rate and calories burnt. It will be a tough one.
