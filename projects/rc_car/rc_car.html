Remote Control Car

If no one will buy me one, I'll make it myself

I never owned a remote control car. Always wanted it but never got it. Don't get me wrong, my childhood did not lack much; I just never got that Tycho Scorcher. Since I got my 3D printer, I have been trying to create projects that intertwined printing and coding. This was my opportunity!
The main purpose of this project is to dust off my CAD design, try new printing designs, improve my coding, and learn about hardware and bus communications.
I will develop this project in several MVPs, building off from the previous one, and improving my design. I'd like to design and print all car parts, and code all logics for the car. Although there are several plug-and-play circuit boards to control the car components, I will only use a Raspberry Pi.
You can access my STL files for the chassis in my Github repo.
